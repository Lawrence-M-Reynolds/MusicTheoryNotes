6 semitones apart.
This is the same as two stacked minor 3rd intervals. 

It naturally occurs on the $ii^\circ$ of the minor key, or the $vii^\circ$ of the major key and then adding the minor 7th:

B - (m3) - D - (m3) - F - (m3) - Ab

But it's also possible to get it by introducing accidentals. e.g. the raised 7th of the minor scale:

E - (m3) - G# - (m3) - B - (m3) - D - (m3) - F

The inversion is also a tritone except that it's spelled differently. This results in two options:
## Diminished 5th (e.g. B-F)
This has a feeling as though it wants to collapse, because:
B-> C
and
F -> E

Either both can move in this way, resulting in:
if only one moves:
P4 (C-F or B - E)
or, if only one moves
Major 3rd (C-E )

But it is quite common to increase the size of one of the jumps by sharpening or flattening..
## Augmented 4th (e.g. F-B)
This has a feeling of expanding, for the same movements as above but in the opposite directions
P5 
or
Minor 6th

The accidentals can also be introduced here in the same way to emphasise the exapnsion.

# Tritone from B-F

| Case   | Resulting Interval | **B Resolution** | **F Resolution**             |
| ------ | ------------------ | ---------------- | ---------------------------- |
| 1, 0   | P4                 | B                | E                            |
| 0,1    | P4                 | C                | F                            |
| 1,1    | M3                 | C                | E                            |
| 2,0    | M3                 | C#               | F                            |
| 0,2    | M3                 | B                | Eb                           |
| 2,1    | m3                 | C#               | E                            |
| 1,2    | m3                 | C                | Eb                           |
| 2,2    | M2                 | C#               | Eb                           |
Note that the last case is very rare as the idea behind a tritone is that it should resolve to a stable harmony. M2 is dissonant like the tritone so it's not a good resolution.

# Picardy Third
There is an interesting case in the $V^7$ chord in the minor scale - **E7** ($E - G\# - B - D$).

To stay in the key, D must resolve by two semitones. But you can still drop by a semitone, which would result in C#. Interestingly you've descended to the raised tone.
